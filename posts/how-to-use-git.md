## How to Use Git — Basic Workflow

Common commands to start tracking your project.

### Setup

1. `git init` — create a repo
2. `git add .` — stage changes
3. `git commit -m "Initial commit"` — commit changes

### Collaboration

- `git branch` and `git checkout -b feature` to create branches.
- `git push` and `git pull` to sync with remotes.

### Tips

- Write clear commit messages.
- Use branches for features and reviews.
