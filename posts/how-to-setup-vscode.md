## How to Set Up VS Code

Optimize VS Code with essential settings and extensions.

### Essentials

- Install extensions: Prettier, ESLint (or Go tools), GitLens.
- Enable auto-save and format-on-save if desired.

### Recommended Settings (example)

- `editor.tabSize`: 2 or 4
- `files.autoSave`: "onFocusChange"

### Tips

- Use workspaces for per-project settings.
- Configure debuggers for your language runtime.
