## How to Create a Strong Passphrase

Use length and unpredictability for secure, memorable passphrases.

### Guidelines

- Use 4+ random words (e.g., 'correct-horse-battery-staple').
- Include capitalization and punctuation if required.

### Tips

- Use a password manager to store long passphrases.
- Avoid predictable phrases or personal information.
