## How to Optimize Images for the Web

Right formats and sizes improve performance.

### Steps

- Choose the right format: WebP/AVIF for photos, SVG for icons.
- Resize images to the display size.
- Compress using tools like `imagemin` or `squoosh`.

### Tips

- Use lazy-loading and responsive `srcset`.
- Cache images with proper headers.
